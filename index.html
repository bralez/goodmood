Welcome to GoodMood.

Goodmood changes your desktop wallpaper with the "Photo of the day" taken from the National Geographic WebSite

Its focus is beauty and simplicity. Goodmood can be started automatically with Windows and it seats quietly on the tray area. It even supports suspend and resume!

Help & Configuration

You can access all the relevant options from the icon on the upper left edge of the main window. A simple menù is shown with the following items:

menù item description
Refresh Get the new "Photo of the day" from the National Geographic Site
Set as desktop background Set the current picture as the window desktop background
Settings -> Automatically check for new pictures Check this item if you want Goodmood to check the National Geographic site periodically
Settings -> Automatically set picture as desktop background Check this item if you want Goodmood to set the picture as windows desktop background after download
Settings -> Run on windows startup Check this item if you want Goodmood to start with Windows
About... Open an about window with version and copyright information
Exit Close Goodmood
