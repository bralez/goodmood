    Welcome to GoodMood.

    Goodmood change you desktop background with the "Photo of the day" picture taken from the National Geographics WebSite

    Its focus is beaty and simplicity. Goodmood can be started automatically on windows startup and it seats quietly on the tray area. It even supports suspend and resume!

    Help & Configuration

    You can access all the relevant options from the icon on the upper left edge of the main window. A simple menù is shown with the following items:

    menù item description
    Refresh Get the new "Photo of the day" from the National Geographics Site
    Set ad desktop background Set the current picture as the window desktop background
    Settings -> Automatically check for new pictures Check this item if you want Goodmood check the National Geographics site periodically
    Settings -> Automatically set picture as desktop background Check this item if you want Goodmood set the picture as windows desktop background after download
    Settings -> Run on windows startup Check this item if you want Goodmood start with Windows
    About... Open an about window with version and copyright information
    Exit Close Goodmood
